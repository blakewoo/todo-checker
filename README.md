# Todo checker

## Overview
It's a web application you can use if you want to observe and check what someone is doing.

## Functions
- Sign up, Sign in, Withdrawal
- TODO : CRUD, Daily, Weekly, Monthly
- Assign a observer
- Realtime chatting with observer

## Tech stack
- nodejs
- express
- ejs
- html
- css
- socket.io
- jest
